AppleScript 学习笔记

2024-09-02 Peritia►Programmatio AppleScript, Programmatio, macOS Comments

第一章：概述与环境

1. AppleScript 简介

  • 脚本语言，专为 macOS 设计。
  • 主要用于自动化任务和控制应用程序。

2. 开发环境配置

  • 使用 Script Editor 进行编写和调试。
    • 安装与访问：Script Editor 是 macOS 内置工具，通常无需额外安装。可通过 Spotlight 搜索 “Script Editor” 进行访问。

3. 基本操作

  • 创建新脚本：打开 Script Editor，选择 新建文稿。
  • 保存脚本：文件 -> 保存，选择 .scpt 格式。
  • 运行脚本：点击工具栏的 运行 按钮。

第二章：基本语法

1. 变量定义

  • 定义变量：使用 set 语句，不需要显示声明类型

2. 数据类型

  • 文本（Text）：字符串，用双引号表示。

  • 数字（Number）：整数或小数。

  • 布尔（Boolean）：true 或 false。

  • 列表（List）：用 {} 表示的有序元素集合。

  • 记录（Record）：键值对集合，用 {} 表示，键和值用 : 分隔。

  • 别名（Alias）：表示文件或文件夹路径。

3. 操作符

  • 算术操作符：+、-、*、/、^

  • 比较操作符：=、≠、<、>、≤、≥

  • 逻辑操作符：and、or、not

4. 条件语句

  • if…then…else：条件判断语句。

5. 循环语句

  • 固定次数循环：repeat + 次数。

  • 带递增变量的循环：使用 repeat with i from。

  • 条件循环：repeat until + 条件。

  • 遍历集合：repeat with item in 集合。
