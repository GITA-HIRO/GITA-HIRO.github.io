使役助动词 せる、させる（しめる）

せる、させる意义

  • 主动的使役，让他者做某事：
    • 先生は学生に本を読ませたり、字を書かせたりする。
    • 教授は助手に学生を教えさせる。
    • 母は子供を使いに行かせる。
  • 表示引起（诱发）：
    • 私たちが訪ねてきたことはよほど主人の心を悦ばせたらしい。
    • 春に入って、淋しく暖かい雨垂の音を聞いていると、なんとなく春の近づくことを思わせる。
    • 校長の話を聞いて、私はすっかり考えさせられてしまった。
  • 容许，放任（或不可抗拒）：
    • 今日は休ませてもらいたい。
    • 失礼させていただきます。
    • あんな映画を子供に見させていいのですか。
    • 彼に勝手に言わせて置くさ。
    • 長女を早く死なせてから臆病になった。

使役句构成

  • 动词为自动词时，被使役者一般用を：
    • 今日は彼を休ませようか。
      有时是に。
    • 夜警に校舎を回らせる。
  • 动词为他动词时，被使役者用に：
    • 父は花子に英語を太郎から習わせた。
  • をして、に命じて：文语用法，略。

せる、させる接续于未然形

  • せる：事实上，一般只有五段。
    • 五段：読む → 読ませる
    • サ变：する → せさせる → させる
  • させる：其他。
    • 上一段：起きる → 起きさせる
    • 下一段：食べる → 食べさせる
    • カ変：来る → 来させる
    • サ変：する → せさせる → させる
      （↑一般必须约音↑）

（しめる为文语残留；意义、接续法等与前二者大致相同。）

    • 今日は彼を休ませようか。
    • 1時間も待てされて、本当にイライラさせられたわ。

    • そろそろ失礼させていただきます。

  • 连用形：

  • 假定形：
    • 警察に調べさせればわかるでしょう。
  • 命令形：
    • 家の中にいる人に戸をあけてさせろ。

敬语/被动/可能(自发)助动词 れる、られる；敬语ます

敬语助动词意义

  • れる、られる：叙述别人动作。
    • あなたが先日言われた本はこれでしょう。
    • 旦那様は家で好んで和服を着られます。
    • 昨日引っ越して来られた鈴木さんです。
  • ます：礼貌、郑重地与他人讲话。
    • 分かりますか。
    • これは私のではありません。

被动助动词意义

  • 直接承受动作，“遭受”、“受到”、“被”:
    不利情况居多。
    • 山田さんは時々先生に叱られる。
    • 私はみんなに笑われました。
    • 田中さんは先生に褒められた。
    • 私は友達に仕事を頼まれた。
  • 间接承受某动作，或受到其影响：
    主要为受害。
    • 今度の旅行は雨に降られてしまってつまらなかった。
    • あの人も息子に死なれてから急に老けた。
    • 夜は子供に泣かれてよく眠れなかった。
  • 以无生物为主语：
    不带感情与利害色彩。古日语少见，受西欧语言影响的近代用法。
    • これは外国人によって設計された建物だ。

被动句构成

  • 主宾易位句：主动者格助词用に、から。
    • 犯人は警察に射殺された。
      犯人 was shot by 警察.
    • 彼は家族のものからも嫌われている。
      彼 is disliked by 家族のもの.
  • 有补语：
    • 彼はいつも友たちに節約することを勧められる。
    • 彼の財布はスリに取られた。（用法不合习惯）
    • 彼はスリに財布を取られた。
    • 妹は母に日記を読まれた。
  • 自然表明句：
    • 空地に家が建てられた。
    • オリンピックは四年ごとに行われる。
    • 卒業試験は二月末日に行われます。
    • 机の上に花瓶が置かれている。
  • 施动者：に、から、によって、で
    • 昨夜一晩中、子供に泣かれて一睡もできなかった。
    • 感謝状が知事から彼に贈られた。
      知事 → 彼
    • 話し合いによって、紛争が解決された。
      The conflict was resolved through discussion.
    • 豪雨で橋が流されてしまった。

可能助动词意义

  • 能力、可能性：
    • この山なら私にも登れよう。
    • 今日は都合が悪くて行かれません。
    • 隣の部屋がうるさくて、なかなか眠られない。
    • この問題全部こたえられれば良いがな。
    • 明日は来られる人だけで良い。
  • 表示自发：
    • なんだか不思議に思われる。
    • 外地の娘のことが案じられる。

可能助动句构成

  • 表可能的对象：が、は（を）。
    • 日本語が話せる（話させる）。
    • これは食べられない。
    • 病気を治せる薬。
      动作主体用には、にも、は、も。
    • 私は日本語が話せる。
    • 彼にも日本語が話せる。
  • 表达可能的方式：
    • れる、られる：
      • 今日は都合が悪くて行かれません。
    • 可能动词：五段未然+れる，并约音。
      • 読む → 読まれる → 読める
      • まだこんな難しい本を読めるところまでいっていない。
    • ことができる：
      • あなたは自転車に乗ることがだきますか。
    • できる、見える、聞こえる、分かる：
      • あなたは自転車がの運転ができますか。
      • 富士山が見える/。
      • 小さい音が聞こえる。
      • 子供には分かるはずはない。
    • 动词连用形+うる(える)：
      • 太陽は西から登ることはありえない。
    • 不能：动词连用形+かねる/わけにはいかない
      • 今すぐお返事致しかねます。
      • こんな天気の船を出すわけにはいきません。
      • 今日は忙しいので、遊んでいるわけにはいかない。

れる、られる接续于未然形

  • れる：
    • 五段：叱る → 叱られる
    • サ変：する → せられる → される
      （↑一般必须约音↑）
  • られる：
    • 上一段：見る → 見られる
    • 下一段：攻める → 攻められる
    • カ変：来る → 来られる
      （↓特别地↓）
    • 使役：
      • せる：行かせる → 行かせられる → 行かされる
        （↑一般必须约音↑）
      • させる：起きさせる → 起きさせられる

否定助动词 ない、ぬ（ん）（しめる）

ない、ぬ（ん）意义

  • 否定：
    • 僕はさっぱり分からない（ぬ/ん）。
    • 教室には誰もいない。
    • どこにも行きません。
  • 婉转劝诱、请求与希望：
    • よかったら遊びに来ない（か）？
    • ご一緒にお茶を飲みません（か）？
    • 早く春が来ないかなあ。
  • ないで表示婉转地禁止：
    • あまり早く歩かないで（ね）。
    • 僕の机に触らないで（よ）。

ない、ぬ（ん）接续于未然形

  • 五段：行く → 行かない/行かぬ（ん）
  • 上一段：見る → 見ない/見ぬ（ん）
  • 下一段：食べる → 食べない/食べぬ（ん）
  • カ変：来る → 来ない/来ぬ（ん）
  • サ変：する → しない/せぬ（ん）
  • 助動詞：
    • 同下一段：
      • （さ）せる → （さ）せない/（さ）せぬ（ん）
      • （さ）せる → （さ）せない/（さ）せぬ（ん）
    • 特殊：
      • ます → ませぬ（ん）
    • etc.

ない活用

活用法 基本形 未然形 连用形 终止形 连体形 假定形 命令形
形容词 ない なかろ なく ない ない なけれ /
なかっ

  • 未然形：

    • 教室には誰もいなかろう（ないだろう）。

  • 连用形：

    • 都合が悪いなら、行かなくてもかまいません。

  • 终止形：

    • 明日は早く起きなければならない。
    • そろそろ帰らないと、暗くなるよ。

  • 连体形：

    • 英語で話せないのは残念だ。

  • 假定形：

    • 戸が開かなければ、中には入れない。

      ない惯用句型

  • にすぎない：“不过…”；“只是…”

    • 今の話は、ただ私の希望にすぎないのです。

  • かも知れない：“说不定…”；“也许…”

    • 明日は雪かも知れない。

  • ...わけにはいかない：“不能…”；“不可以…”

    • 今日は忙しので、遊んでいるわけにはいかない。

  • ...にほかならない：“无非是…”；“不外乎是…”

    • 今日の成功を見たのは絶えない努力の結果にほかならない。

  • ...てたまらない/てはならない/てやまない：“…（得）不得了”；“…（得）很”

    • 大学試験に合格したときは嬉しくてたまらなかった。
    • 母の病気が心配でならない。
    • 私はその日の一日も早く訪れるのを心からお願ってたまない。

  • ...てはいけない/てはならない：“不许…”；“不得”

    • 暴力をふるってはいけないね。
    • これは子供が見てはならないものです。

  • なければならない（なければいけない、なくてはならない、なくてはいけない）：“必须…”；“不…（就）不行”

    • 早く寝なくてはいけません。

特殊 ぬ（ん） / ず ぬ ぬ ね /
  • 连用形：
    • たまには飲まずに帰って来てください。
    • 万里を遠しとせずして中国に来られた。
  • 终止形：
    • 火のない所は煙は立たぬ。

    • 分からぬ（ん）ことは尋ねるがよい。

ず活用

活用法 基本形 未然形 连用形 终止形 连体形 已然形 命令形
文语 ず ず ず ず ぬ ね ざれ
ざら ざり ざる ざれ
ざり系列，由ず+あり约音得到。
  • 未然形：
    • 知らずば、言って聞かせよう。
  • 终止形：
    • 無用のもの入るべからず。
    • 百聞は一見に如かず。
  • 连体形：
    • はなやかな勝利のかげ、知られざる悲劇があった。
  • 已然形：
    • 急がねば間に合うまい。

ず惯用句型

  • ...ずにはいらない：“不由得…”，“不禁…”，“忍不住要…”

    • 全て笑わずにはいられない。

  • ...ずにはおかない：“势必…”，“不得不…”

    • 彼の演説は各界の人反響をよびおこさずにはおかない。

  • ...ざるをえない：“不得不…”，“只得…”

    • …認めざるをえない。
    • …物価安定を急がざるをえない。

なくて、ないで

有时二者区别并不明确。
  • なくて：
    • 因果：
      • 雨が降らなくて助かった。
      • 病気に治らなくて心配した。
    • 连接并列（？）事项：
      • あの人は何もしなくて、みんなやったのは私です。
      • 今までのは序論にすぎなくて、これからが本論です。
    • 惯用句型：
      • もういかなくていいです。
      • 都合が悪ければ、いかなくても構いません。
      • そうしてなくてはならない（いけない）。
  • ないで：
    • 同一主体，“不…，而…”。后项为意志性而非状态性动作。
      • 誰にも言わないでこっそり逃げた。
      • 今日も授業をしないで、映画を見せた。
      • ケーキを買わないで家に帰りました。
    • “请不要…”，“请别…”。
      • ゴミを捨てないでください。
      • 今は何も聞かないで。
    • 某种状态。
      • 彼はまだ起きないでいる。
      • 知りもしないでペラペラ喋る。

希望助动词たい、たがる

たい意义

心情上的希望。
  • 讲话人心中的希望。
    • 今日は早くうちへ帰りたい。
    • 今は誰も会いたくない。
  • 以提问、推测、传闻、假定、样态、说明的方式涉及他人希望。
    • 君も行きたいか。
    • 彼も行きたいでしょう。
    • 彼を行きたいと言ったね。
    • 行きたければ、はっきり言うがいい。

たい接续于连用形

  • 五段：行く → 行きたい
  • 上一段：見る → 見たい
  • 下一段：受ける → 受けない
  • カ変：来る → 来たい
  • サ変：する → したい
  • 助動詞：
    • 下一段：
      • （さ）せる → （さ）せたい
      • （ら）れる → （ら）れたい
    • etc.

    • 田中さんがあなたに頼みたいんです。
  • 假定形：
    • 君は帰りたければ、帰ってもいいよ。

...が...たい / ...を...たい

  • ...が...たい：单纯、本能的愿望。
    • 水が飲みたい。
    • 小説が読みたい。
  • ...を...たい：复杂、理性场合；句子成分复杂。
    • レコードを一、二枚買いたいのです。
    • 水を手で掬って飲みたい。

たがる意义

表露在言行上，可以察觉的希望
  • eg.
    • とても素晴らしい映画なので、彼も見たがっている。
    • あまり行きたがるものだから、つい連れて行く気になった。
  • 非某一时候，而是某种习惯。
    • 裏話を聞きたがる。
    • 子供は飴ばかり食べたがるから、なれべく見せないようにしている。
  • 条件、定语、传闻、推断、说明、会议等，可以用于一、二人称。
    • 君も行きたがっているそうだ。
    • 私が行きたがったているのに、あなたは知らん顔をしている。

た意义

  • 过去：
    • この間の映画はとても面白かった。
    • 私は富士山を見たことがある。
  • 完了：
    • ああ、これでやっと仕上がった。
    • もうすっかり治ったから、明日は学校に行けるぞ。
    • 借りたものは返さねばならぬ。

  • 未然形：

  • 连用形：

    • あの頃は私はまだ学生だった。
    • 集会の場所はクラブだっ（でし）たりレストランだっ（でし）たり定まっていません。
    • 吾輩は猫である。

    • 今日は労働感謝の日で（でして）、学校は休みです。

  • 终止形：

    • 今日は日曜日だ（です）。
    • 会議は明日の午後だそうだ。
    • 今日はとても暑いです。

  • 连体形：

    • それは何よりの証拠なのだ。
    • 発展途上国なので、なんの心配もない。
    • 秋山は級長なはずはない。
    • あれは山なものか、勿論雲さ。
    • 準備不足ですので、自信がありません。

  • 假定形：

    • 明日雨なら会場を室内に移そう。
    • やってみたいなら、やってみたまえ。

だ、です接续于体言等

  • 接续于体言、副词、某些助词（の、から、まで、ばかり、だけ、ぐらい、ほど、など、きり、ずつ）。
    • 体言：
      • 学生 → 学生だ（です）
    • 副词：
      • 大変 → 大変だ（です）
    • 助词：
      • （東京）から → （東京）からだ（です）
  • 各活用形接续法：
    • 未然形だろう、でしょう（う）接续于终止形（动词、形容词、部分助动词）。
      • 雨がやむだろう（でしょう）。
      • 広州はもっと暑いだろう（でしょう）。
      • 笑われるだろう（でしょう）。
    • 假定形なら接续于活用词（动词、形容词、部分助动词）。
      • 君が行くなら、僕も行こう。
      • 面白くないならやめましょう。
      • 先生がいらっしゃるなら、私もお伴いたします。
      • やってみたいなら、やってみたまえ。
    • です可以接续于たい、らしい、た、ん后面构成敬语，だ无此用法。
      • 面白いです。（面白かったです。）
      • 読みたいです。
      • 面白いらしいです。
      • 雨は降りませんでした。
  • 注意：
    • だ、です不能接续于助动词う、よう、まい。
    • 形容动词（静かだ（です））、助动词（ようだ（です）、そうだ（です））的词尾だ（です）不是断定助动词。

推量助动词

う、よう意义

  • 表示推量。
    • 明日あたりから天気がよくなろう。
    • 今も丈夫だろうと思います。
    • 間も無く空も晴れよう。
  • 表意志（主要是说话人的）。
    • 頭が痛いから、学校を休もう。
    • 明日の朝は早く起きよう。
  • 表示劝诱、希望、号召，主语包括讲话人在内
    • いっしょに学校へ行こう。
    • あの映画を見ましょうね。
    • みんなで相談しようじゃないか。
  • う（よう）とする：意志（可表示他人）；动作、作用、状态即将实行或实现。
    • 手紙を書こうとしたが、ペンがないのでやめた。
    • 出かけようとして玄関まで出てきたら、田中さんがたずねてきた。
    • あの人は夜になっても帰ろうとしなかった。
    • 電車のドアがしまろうとしたとき、男の子がとびのった。
    • 夜があけようとするころやっと仕事がおわった。
    • 梅の花がいまにも開こうとしている

う、よう活用词形不变

  • 终止形：

    • 夕方から雨が降るだろう。
    • あの大きな声で呼ばれたら、目もさめよう。
    • 来春は大学入試を受けようとしている。
    • 苦痛であろうが、がまんしてもらいたい。

  • 连体形：
    一般只接续はず、こと、もの等形式体言。

    • あの人がそんな悪い事などしようはずがない。
    • 勉強しながらいねむりしようものなら、ひどくしかられるよ。

      う、よう的惯用句型

  • ...う（よう）じゃないか：号召、希望，“让（我们）…吧”、“（我们）该…啦”

    • ぼつぼつ会議をはじめようじゃないか。

  • ...う（よう）が...う（よう）が...。、...う（よう）と...う（よう）と...。：”（无论）…也好，…也好…。”

    • 雨が降ろうが（と）やりが降ろうが（と）、行かなければな
      ちない。
    • 雨が降ろうと、風が吹こうと、毎日出かけて行きました。

  • う（よう）が（と）...まいが（と）：参照まい。

  • う（よう）ものなら...。：如果…，那可就…”。

    • こんな汚いものを食べようものなら、すぐ病気になるよ。
    • あの人にそんなことを言おうものなら，（君はあの人に）なぐられるよ。

  • うに（主要为こともあろうに、人もあろうに）：谴责、责怪。

    • 雨が降ったらこまろうに，あの子は傘を持たずに出かけた。
    • あの人はこともあろうに，人の家に火をつけたのですよ。
    • あの人は人もあろうに、あんなばかな女と結婚したのよ。

まい意义

  • 否定的推量：

    • まさかそんなこともあるまい。
    • この分では明日も晴れまい。
    • 彼も賛成しますまい。
    • 彼はおそらく承知しまい。

  • 否定的意志：

    • あの人にはもう二度と頼むまいと決心した。
    • これからは映画を見まいと思った。
    • 私は今後彼と交渉しますまい。
    • こんな誤りは二度としまいと思っていた。

  • 否定的劝诱：

    • これからお互いに冗談を言うまい。
    • すぎさったことはくよくよ考えますまい。
      接续ぞ表示禁止
    • そんなことはいうまいぞ。
    • 二度と繰り返すまいぞ。

      まい接续法

  • 终止形：

    • 五段：降る → 降るまい
    • ます:（見）ます → （見）ますまい

  • 连体形：

    • 上一段：見る → 見まい
    • 下一段：受ける → 受けまい
    • カ変：来る → 来まい
    • サ変：する → しまい
