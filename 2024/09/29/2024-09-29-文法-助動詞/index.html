使役助动词 せる、させる（しめる）

使役助动词意义

  • 主动的使役，让他者做某事：
    • 先生は学生に本を読ませたり、字を書かせたりする。
    • 教授は助手に学生を教えさせる。
    • 母は子供を使いに行かせる。
  • 表示引起（诱发）：
    • 私たちが訪ねてきたことはよほど主人の心を悦ばせたらしい。
    • 春に入って、淋しく暖かい雨垂の音を聞いていると、なんとなく春の近づくことを思わせる。
    • 校長の話を聞いて、私はすっかり考えさせられてしまった。
  • 容许，放任（或不可抗拒）：
    • 今日は休ませてもらいたい。
    • 失礼させていただきます。
    • あんな映画を子供に見させていいのですか。
    • 彼に勝手に言わせて置くさ。
    • 長女を早く死なせてから臆病になった。

使役句构成

  • **动词为自动词时，被使役者一般用を**：
    • 今日は彼を休ませようか。
      有时是に。
    • 夜警に校舎を回らせる。
  • **动词为他动词时，被使役者用に**：
    • 父は花子に英語を太郎から習わせた。
  • **をして、に命じて**：文语用法，略。

使役助动词せる、させる接续于未然形

  • **せる**：事实上，一般只有五段。
    • 五段：読む -> 読ませる
    • サ变：する -> せさせる -> させる
  • **させる**：其他。
    • 上一段：起きる -> 起きさせる
    • 下一段：食べる -> 食べさせる
    • カ変：来る -> 来させる
    • サ変：する -> せさせる -> させる
      （↑一般必须约音↑）

（しめる为文语残留；意义、接续法等与前二者大致相同。）

  • 未然形：
    • 今日は彼を休ませようか。
    • 1時間も待てされて、本当にイライラさせられたわ。
  • 连用形：
    • そろそろ失礼させていただきます。
    • 僕は彼を行かせたい。
  • 终止形：
    • 子供を一人で寝させる。
    • 今度は女性を行かせるそうです。
  • 连用形：
    • あの人にその仕事を続けさせることはできません。
  • 假定形：
    • 警察に調べさせればわかるでしょう。
  • 命令形：
    • 家の中にいる人に戸をあけてさせろ。

敬语/被动/可能(自发)助动词 れる、られる；敬语ます

敬语助动词意义

  • **れる、られる**：叙述别人动作。

    • あなたが先日言われた本はこれでしょう。
    • 旦那様は家で好んで和服を着られます。
    • 昨日引っ越して来られた鈴木さんです。

  • **ます**：礼貌、郑重地与他人讲话。

    • 分かりますか。
    • これは私のではありません。

      被动助动词意义

  • 直接承受动作，“遭受”、“受到”、“被”:
    不利情况居多。

    • 山田さんは時々先生に叱られる。
    • 私はみんなに笑われました。
    • 田中さんは先生に褒められた。
    • 私は友達に仕事を頼まれた。

  • 间接承受某动作，或受到其影响：
    主要为受害。

    • 今度の旅行は雨に降られてしまってつまらなかった。
    • あの人も息子に死なれてから急に老けた。
    • 夜は子供に泣かれてよく眠れなかった。

  • 以无生物为主语：
    不带感情与利害色彩。古日语少见，受西欧语言影响的近代用法。

    • これは外国人によって設計された建物だ。

被动句构成

  • 主宾易位句：主动者格助词用に、から。

    • 犯人は警察に射殺された。
      犯人 was shot by 警察.
    • 彼は家族のものからも嫌われている。
      彼 is disliked by 家族のもの.

  • 有补语：

    • 彼はいつも友たちに節約することを勧められる。
    • 彼の財布はスリに取られた。（用法不合习惯）
    • 彼はスリに財布を取られた。
    • 妹は母に日記を読まれた。

  • 自然表明句：

    • 空地に家が建てられた。
    • オリンピックは四年ごとに行われる。
    • 卒業試験は二月末日に行われます。
    • 机の上に花瓶が置かれている。

  • 施动者：に、から、によって、で

    • 昨夜一晩中、子供に泣かれて一睡もできなかった。
    • 感謝状が知事から彼に贈られた。
      知事 -> 彼
    • 話し合いによって、紛争が解決された。
      The conflict was resolved through discussion.
    • 豪雨で橋が流されてしまった。

      可能助动词意义

  • 能力、可能性：

    • この山なら私にも登れよう。
    • 今日は都合が悪くて行かれません。
    • 隣の部屋がうるさくて、なかなか眠られない。
    • この問題全部こたえられれば良いがな。
    • 明日は来られる人だけで良い。

  • 表示自发：

    • なんだか不思議に思われる。
    • 外地の娘のことが案じられる。

可能助动句构成

  • 表可能的对象：が、は（を）。

    • 日本語が話せる（話させる）。
    • これは食べられない。
    • 病気を治せる薬。
      动作主体用には、にも、は、も。
    • 私は日本語が話せる。
    • 彼にも日本語が話せる。

  • 表达可能的方式：

    • **れる、られる**：
      • 今日は都合が悪くて行かれません。
    • 可能动词：五段未然+れる，并约音。
      • 読む -> 読まれる -> 読める
      • まだこんな難しい本を読めるところまでいっていない。
    • **ことができる**：
      • あなたは自転車に乗ることがだきますか。
    • **できる、見える、聞こえる、分かる**：
      • あなたは自転車がの運転ができますか。
      • 富士山が見える/。
      • 小さい音が聞こえる。
      • 子供には分かるはずはない。
    • **动词连用形+うる(える)**：
      • 太陽は西から登ることはありえない。
    • 不能：动词连用形+かねる/わけにはいかない
      • 今すぐお返事致しかねます。
      • こんな天気の船を出すわけにはいきません。
      • 今日は忙しいので、遊んでいるわけにはいかない。

        れる、られる接续于未然形

  • **れる**：

    • 五段：叱る -> 叱られる
    • サ変：する -> せられる -> される
      （↑一般必须约音↑）

  • **られる**：

    • 上一段：見る -> 見られる
    • 下一段：攻める -> 攻められる
    • カ変：来る -> 来られる
      （↓特别地↓）
    • 使役：
      • **せる**：行かせる -> 行かせられる -> 行かされる
        （↑一般必须约音↑）
      • **させる**：起きさせる -> 起きさせられる
